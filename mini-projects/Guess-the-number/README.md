# Guess the number

#Как он работает:

Компьютер выбирает случайное число ( от 0 до 10).  
Игрок задаёт своё число тоже от 0 до 10. 
Эсли в результате числа совпадают в консоле будет написано: "Ты угадал число". Если нет – ты не угадал.  
В результате ты имеешь ещё второй шанс, если и здесь не угадаешь то тебе напишет: "Ты проиграл, игра окончена, число компьютера было ...".
